# SAT Readiness Checklist

Use this checklist before site acceptance testing for a BESS installation.

## Site Readiness

| Item | Evidence Required | Status |
|---|---|---|
| Installation complete | Mechanical, electrical, and communication installation records | Not started |
| Energization permit | Approved site permit and switching plan | Not started |
| Grounding verified | Ground continuity and resistance test records | Not started |
| Fire safety interface checked | Fire alarm, suppression, and emergency stop interface records | Not started |
| Network connectivity confirmed | SCADA, EMS, BMS, PCS, and data gateway communication checks | Not started |

## Commissioning Evidence

| Item | Evidence Required | Status |
|---|---|---|
| Pre-commissioning tests complete | Insulation, continuity, torque, and visual inspection results | Not started |
| Firmware versions recorded | BMS, PCS, EMS, and auxiliary controller versions | Not started |
| Protection settings verified | Approved settings file and field verification record | Not started |
| Functional tests ready | Charge/discharge, alarms, trips, and remote command cases | Not started |
| Performance test plan approved | Test cases, measurement method, and acceptance criteria | Not started |

## Handover Readiness

| Item | Evidence Required | Status |
|---|---|---|
| As-built package prepared | Drawings, test records, manuals, certificates, and punch status | Not started |
| Operator training complete | Training agenda, attendance, and training material | Not started |
| O&M documentation delivered | Maintenance schedule, spare parts, emergency response instructions | Not started |
